# 🎤 Eurovision 2026 – Interaktiv Finale-app

> Laget for å bruke under Eurovision Song Contest 2026-finalen fra Wien.  
> Ingen installasjon, ingen konto – bare åpne i nettleseren og stem!

![Eurovision 2026 Vienna](https://img.shields.io/badge/Eurovision-Vienna%202026-e91e8c?style=for-the-badge)

![License](https://img.shields.io/badge/License-MIT-purple?style=for-the-badge)

## ✨ Funksjoner

### 🎤 Stemmeskjema
- Alle 25 finalister med flagg, artist og sangtittel
- Gi **1–5 stjerner** for fire kategorier: Vokal, Melodi, Styling og Show
- Totalpoenget regnes ut automatisk
- «I ledelsen»-banner som oppdateres live

### 🏆 Toppliste
- Automatisk rangering etter samlet poengsum
- 🥇🥈🥉 medaljer til topp 3
- Poengbar som visualiserer gap mellom land
- Detaljerte stjerneratinger per kategori

### 🎯 Bingo
- 5×5-brett med 24 ting å se og høre under showet
- Gratis midtfelt
- Kryss av mens du ser på
- Konfetti-eksplosjon 🎉 når brettet er fullt

---

## 🚀 Kom i gang

```bash
git clone https://github.com/ditt-brukernavn/eurovision-2026.git
```

Åpne `index.html` i nettleseren – ferdig.

Eller last ned filen direkte og dobbeltklikk på den.

**Ingen server, ingen npm, ingen bygg-steg.**

---

## 📱 Mobilvennlig

Appen fungerer på mobil og nettbrett. Del lenken (eller filen) med venner – alle kan stemme på sitt eget skjema.

---

## 💾 Lagring

Alt lagres automatisk i nettleserens `localStorage`.  
Lukk fanen og åpne igjen – poengene er der.  
Klikk «Nullstill alt» for å starte på nytt.
